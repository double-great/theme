# great-great-jekyll-theme

Jekyll theme for [doublegreat.dev](https://doublegreat.dev).

## Installation

Add this line to your Jekyll site's `_config.yml`:

```yaml
theme: great-great-jekyll-theme
```

And add to the `plugins` list:

```yaml
- jekyll-remote-theme
```
